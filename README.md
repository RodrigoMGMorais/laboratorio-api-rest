# Laboratório API REST
Meu projeto de estudo: API REST simples desenvolvida em Python com Flask para gerenciamento de usuários.

---

## 📋 Funcionalidades
- ✅ Listar todos os usuários
- ✅ Consultar um usuário por ID
- ✅ Adicionar novo usuário
- ✅ Atualizar dados de um usuário
- ✅ Alterar ID do usuário
- ✅ Excluir usuário
- ✅ Dados salvos em arquivo JSON (não somem ao reiniciar)

---

## 🚀 Tecnologias utilizadas
- Python 3
- Flask
- JSON para armazenamento
